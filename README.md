## Description

This repository contains the data processing scripts and code used in the study:

**Weather Impacts and Input Adaptation: Farm-Level Evidence on Corn and Wheat Yields in Italy**\
Authors: Giacomo Coughlan, Paolo Nota, Daniele Cavicchioli, Alessandro Olper

The scripts and documentation are provided for the purpose of transparency and reproducibility, and correspond to the version of the analysis accepted for publication. Upon publication of the paper, a full citation and direct link to the article will be added here.

## Contents

-   R scripts for data processing, analysis, and projections
-   Directory structure and scripts to generate results
-   A script is provided to generate synthetic data replacing non-shareable microdata. Publicly available data are included.
